# Бот «Точно не новости»

Telegram-бот с тестами по медиаграмотности для проекта «Точно не новости».

## Что умеет бот

- Тесты по 5 темам, два уровня сложности (после видео / после теории)
- 5 вопросов на каждый уровень с объяснениями при ошибке
- Угадайка: Вестник или Глашатай?
- Сохранение результатов пользователя

## Деплой на Railway

1. Создайте репозиторий на GitHub и загрузите эти файлы
2. Зайдите на railway.app и войдите через GitHub
3. Нажмите New Project → Deploy from GitHub repo
4. Выберите ваш репозиторий
5. В разделе Variables добавьте переменную:
   - BOT_TOKEN = (ваш токен от BotFather)
6. Railway автоматически развернёт бота

## Локальный запуск (для теста)

```bash
pip install -r requirements.txt
BOT_TOKEN=ваш_токен python bot.py
```
